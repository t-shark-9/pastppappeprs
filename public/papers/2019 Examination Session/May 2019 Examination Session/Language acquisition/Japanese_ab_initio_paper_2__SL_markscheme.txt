                                          Section A
Criterion A: Language

• Does the student demonstrate an ability to use the language effectively and accurately?
• Does the student use spelling, calligraphy (see “Glossary of terms”), vocabulary and
  grammatical structures correctly and accurately?

  Marks    Level descriptor
    0      Language inaccuracies completely obscure communication.
    1      Language inaccuracies often obscure communication.
    2      Language is generally accurate and does not obscure communication.
    3      Language is mostly accurate and communication is clear.


補足説明

字数
最低字数は 100 字。セクション A では、最低字数に満たなくても減点しない。

言語力
間違いの程度による。意味が通じる場合やうっかりミスは、問題ない。意味が通じない場
合や、してはいけない基本的な文法ミスは、減点対象になる。

脱字やつづりのミス
たまに脱字やつづりのミスがある。

字体や動詞の変形
動詞の形態と常体の使い分けができなかったり、動詞の過去形が正しく使えない。

文型
特定の文型が正しく使えない。

  accurately?

           The response and communication are of limited effectiveness.
           The range of vocabulary used is limited.
   1–2     Some basic grammatical structures are used accurately. These are isolated and limited
           in range.
           Register is inappropriate.

           Register is partially appropriate.
           The response is generally accurate and communication is generally effective.
           The range of vocabulary used is varied.

           structures are used accurately.

           Register is appropriate.

Criterion C: Format

• To what extent does the student succeed in using the correct text type?
• Does the student use the appropriate format?

  Marks    Level descriptor
    0      The format is not appropriate.
    1      The format is partially appropriate.
    2      The format is appropriate.

テキストタイプに必要な下記の事柄 3 つの中で 2 つ書けていて 2 点を与える。
1 つ目の必須事項が無いと 1 点。

必要な事柄は次のとおり。

Question 3: 新聞記事
   1. 見出し/タイトル(必須)
   2. 自分の名前
   3. 読者を意識した文

Question 4: フォーマルな手紙
   1. 相手の名前(会社の人へ)または最初の挨拶(必須)
   2. しめくくりの挨拶
   3. 自分の名前

Question 5: レポート
   1. 見出し/タイトル(必須)
   2. 自分の名前
   3. 感想など効果的なしめくくり
