Teks A — Anda akan mendengar pengumuman di terminal bus.


Juruhebah   Sila ambil perhatian, berikut adalah pengumuman khas dari pihak pengurusan terminal.

            Para tetamu yang dihormati sekalian selamat datang ke Terminal Bas Kuala Helang. Pihak pengurusan ingin merakamkan
            ucapan terima kasih kerana memilih kami sebagai rakan perjalanan anda. Pilihan anda adalah tepat sekali kerana kami akan
            memastikan perjalanan anda satu pengalaman yang tidak dapat dilupakan.

            Pihak kami ingin para tetamu untuk memberi perhatian kepada perkara-perkara berikut. Pertamanya, pihak kami ingin
            meningatkan para tetamu untuk menyimpan tiket di tempat yang selamat. Kehilangan pas masuk ini akan mengakibatkan
            penumpang tidak dibenarkan meneruskan perjalanan.

            Selain itu, tetamu yang membawa anak-anak diminta mengawasi mereka bagi mengelakkan sebarang kecederaan. Ini kerana,
            kawasan terminal mempunyai banyak bas yang bergerak keluar dan masuk.

            Seterusnya para tetamu juga diminta memastikan bagasi anda berada dalam keadaan yang selamat. Pastikan bagasi anda
            tidak mengandungi bahan terlarang yang akan mengakibatkan anda menghadapi tindakan undang-undang. Hukuman berat
            akan dikenakan kepada individu yang didapati melakukan kesalahan membawa bahan terlarang seperti dadah dan senjata
            api yang boleh membawa kepada hukuman yang berat.

            Para tetamu juga diminta bantu memastikan persekitaran terminal berada dalam keadaan bersih. Buang sampah dalam
            bekas-bekas yang disediakan bagi memastikan keselesaan terminal pada sepanjang masa. Terminal bas ini adalah milik kita
            bersama.

            Pihak pengurusan ini mengucapkan selamat jalan dan selamat sampai ke destinasi yang dituju.

Teks C — Anda akan mendengar seorang ahli bomba sedang memberi taklimat kepada pelajar cara menggunakan alat pemadam api.


 Ahmad Fuad Jamil     Selamat sejahtera para pelajar. Saya Ahmad Fuad Jamil dari Unit Pendidikan Cegah Kebakaran. Saya akan
                      menerangkan cara-cara menggunakan alat pemadam api. Mari kita mulakan.

                      Pertama: Kenal pasti laluan untuk melarikan diri dengan selamat
                      Sebelum mengendalikan alat pemadam api, pastikan anda mempunyai laluan keluar yang jelas. Jika tidak dapat
                      memadamkan api, cari jalan keluar yang selamat. Oleh itu, tempat letak alat pemadam api sebaiknya berdekatan
                      jalan keluar.

                      Kedua: Berdiri ke belakang
                      Pastikan anda berdiri mengadap api yang hendak dipadam dan pada masa yang sama pastikan belakang badan
                      anda mengadap laluan keluar. Apabila anda sudah bersedia untuk mengendalikan alat pemadam api pastikan anda
                      berada pada jarak 1.00m atau 1.5m dari api.

                      Ketiga: Menggunakan alat pemadam api
                      Memang sukar untuk berfikir dengan jelas semasa kecemasan, untuk bantu anda ketika kecemasan ingat PASS.
                      Sudah bersedia untuk catat maksud PASS?

                                                                 ***

 Ahmad Fuad Jamil     Baiklah, P.A.S.S ertinya:

                      •   Pull bermaksud Tarik iaitu tarik pin pada alat pemadam api.
                      •   Aim bermaksud Tuju iaitu halakan muncung hos pemadam ke arah dasar api.
                      •   Squeeze bermaksud Tekan iaitu picit pemegang untuk mengeluarkan cecair pemadam api.
                      •   Sweep bermaksud Sapu iaitu pastikan alat pemadam api dihalakan pada pangkal api dan gerakkannya dari sisi
                          ke sisi sehingga api padam.

                      Keempat: Perhatikan kawasan
                      Selepas api kelihatan padam, teruskan memerhati kawasan api untuk memastikan ia tidak menyala semula. Jika api
                      menyala semula, ulangi proses PASS.

Kelima: Hubungi bomba
Apabila api sudah kelihatan padam, jika anda tidak berpeluang menghubungi bomba sebelum ini lakukanlah
sekarang. Mereka akan dapat memeriksa tapak kebakaran dan memastikan ia dipadamkan sepenuhnya.

Keenam: Pergi ke tempat yang selamat
Sebaik sahaja api padam, atau jika anda tidak dapat memadamkan api, tinggalkan tempat kejadian dan cari tempat
lapang yang jauh dari jangkauan api.

Baiklah, itulah langkah-langkah yang akan membantu anda jika berlaku sebarang kebakaran.
